# 플로이드 와샬
- '모든 정점'에서 '모든 정점'으로의 최단 경로를 구하는 알고리즘
- '거쳐가는 정점'을 기준으로 최단거리를 구하는 것
- 다이나믹 프로그래밍 이용

~~~c++
//거점노드
    for(int k=1; k<=V; k++)
    {
        //출발노드
        for(int i=1; i<=V; i++)
        {
            //도착노드
            for(int j=1; j<=V; j++)
            {
                if(dp[i][j] > dp[i][k] + dp[k][j])
                {
                    dp[i][j] = dp[i][k] + dp[k][j];
                }
            }
        }
    }
~~~
